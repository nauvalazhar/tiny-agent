HTML Details Tag Examples

The details tag creates collapsible content sections. Click on the summaries to expand/collapse them.

What is the details tag?

The details tag is an HTML element that creates a disclosure widget where information is visible only when toggled open. It's commonly used for FAQ sections, expandable content, and progressive disclosure.

Features of the details tag
  • Native browser support (no JavaScript required)
  • Accessible by default
  • Can contain any HTML content
  • Keyboard navigable
  • SEO friendly
This section is open by default

You can use the "open" attribute to make a details element expanded by default. This is useful when you want to show important content initially but still allow users to collapse it.

Browser support

Supported in all modern browsers including Chrome, Firefox, Safari, and Edge.

Styling options

You can style the summary element and the content inside details with CSS. The marker can also be customized.

Basic HTML structure

Here's the basic structure:

<details>
    <summary>Click to expand</summary>
    <p>This content is hidden until expanded.</p>
</details>
